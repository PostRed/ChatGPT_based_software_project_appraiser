Оцените качество репозитория со следующим содержимым от 1 до 100.
Опирайтесь на следующие параметры:
1 Качество управления проектами
2 Насколько проект готов принять новых программистов
2.1 Насколько комфортно будет работать новым программистам
2.2 Насколько репозиторий понятен начинающим программистам

Всего строк: $total_lines
Количество дней с момента последнего изменения: $number_of_days_since_last_change
Количество звезд: $stars_count
Количество участников: $count_of_contributors
Количество вилок: $forks_count
Количество открытых вопросов: $count_of_open_issues
Количество закрытых вопросов: $count_of_closed_issue
Количество объединенных пул-реквестов: $count_of_merged_pull_requests
Количество строк комментариев: $count_of_comment_lines
Цикломатическая сложность*: $cyclomatic_complexity
Количество строк комментариев к коммитам: $count_of_commit_comment_lines
Количество файлов с синтаксическими ошибками: $syntax_errors

Метрика цикломатической сложности* (Cyclomatic Complexity) - это количественная мера сложности программного кода, которая оценивает количество линейно независимых путей через код.
